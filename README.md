# Simple Interest Calculator

This project contains a simple interest calculator implemented using a Bash script.

## Description
The calculator computes simple interest based on user-provided input values:
- Principal amount
- Rate of interest
- Time period (in years)

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Usage
Run the script `simple-interest.sh`, then enter the required values when prompted.
